## Requirement
1. Make sure you have installed Node

## Run the game

Run ```npm install``` in order to install all dependencies.
```
$ npm install
```

Run ```npm start``` to start the server and build front-end code.

```
$ npm start
```

Go to ```http://localhost:3000``` to play the game
